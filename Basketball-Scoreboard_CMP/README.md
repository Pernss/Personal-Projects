BASKETBALL SCORBOARD -- WEBSITE

DESCRIPTION:
A simple implemetation of the basketball scoreboard using just html,css and javascript

HOW IT WORKS: 
Buttons will be displayed once you went to the website, just like the normal game of Basketball there will be points wether it's 2 or 3 points and also some fouls for both teams.

You can pause or play the game by clicking the timeout button or play button, you can even reset the game and refreshn the timer.
