CHROME EXTENSION -- LEAD TRACKER

HOW IT WORKS:

This extension is like a simple bookmark feature of Google Chrome, it saves the link that you choose.

You can also input a url and save it into the lead tracker if you remember the url of the website you want to save.

You can also delete all of exisiting lead tracker if you desire.
Note: when you want to delete all of the tracker use double click to implement this feature.
